This is by no means a production ready game, merely something I was playing around with to test out the capabilities of canvas and javascript animation in the browser. There are not even levels, or a boss, merely the shell of some gameplay. I have the code on Github, https://github.com/mycrazydream, or of course you can just view the source. It's all javascript and sprites. Of course it is much easier to grab with git. I've had trouble running it in Firefox on OS X recently as some of the gameplay keys have been taken over by the browser and I haven't fixed it yet. Chrome definitely works though. I will probably improve/add-to it as time goes on.
